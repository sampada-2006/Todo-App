# 📝 Todo App

A simple Todo Application built using HTML, CSS, and JavaScript.

## 🚀 Features
- Add new tasks
- Delete tasks
- Store tasks using localStorage
- Tasks persist after page refresh

## 📂 Project Structure
- ToDoApp.html
- ToDoApp.css
- ToDoApp.js

## 💡 Concepts Used
- DOM Manipulation
- JavaScript Arrays & Objects
- localStorage
